# Stikmontas

Vienpuslapė (one-page) svetainė įmonei **Stikmontas** — stiklo ir aliuminio gaminių gamyba ir montavimas (terasų ir balkonų stiklinimas, dušo kabinos, berėmis stiklas, pertvaros, turėklai, veidrodžiai, stogeliai). Panevėžys, dirbame visoje Lietuvoje. 25 metų patirtis.

## Stack
- Statinis HTML / CSS / JavaScript (be framework'ų)
- Lenis (smooth scroll) + GSAP ScrollTrigger (CDN)
- Šriftai: Cabinet Grotesk + Switzer (Fontshare)
- Realios darbų nuotraukos galerijoje (WebP)

## Vietinė peržiūra
```bash
node server.js   # http://localhost:3032
```

## Struktūra
- `index.html` — visas puslapis
- `assets/css/style.css` — dizaino sistema
- `assets/js/main.js` — navigacija, galerijos filtras + „Rodyti daugiau", lightbox, forma
- `assets/img/` — darbų nuotraukos
- `server.js` — minimalus statinis dev serveris
